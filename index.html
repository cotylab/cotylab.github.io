코티랩 코딩 교육 연구소

코딩 능력은 이제 선택이 아니라 필수 입니다. 대부분의 산업 분야에서 소프트웨어가 차지하는 비중이 높아져 가고, 그 소프트웨어는 코딩으로부터 탄생합니다.
코티랩은 블록 코딩부터 고급 코딩까지 전문적인 맞춤식 코딩 교육을 제공합니다. 개인별 교육 목표를 설정하고 커리큘럼을 설계하여 체계적이고 흥미롭게 코딩을 공부합니다.

블록 코딩 block coding

스크래치 scratch, 엔트리 entry 등의 블록 코딩 도구로 코딩에 필수적인 컴퓨팅 사고력 computational thinking을 학습합니다.

실전 코딩 real coding

파이썬 python, 자바 java 등의 프로그래밍 언어로 객체지향 프로그래밍 object-oriented programming을 학습합니다.
